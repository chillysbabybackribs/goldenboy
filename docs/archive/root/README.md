# Root Archive

This folder contains planning notes, audits, benchmarks, and other historical documents that were moved out of the repository root.

The root directory is intentionally reserved for:

- runtime-used prompts and entrypoint docs such as `AGENT.md`, `CLAUDE.md`, and `README.md`
- code or executable project config such as `package.json`, `tsconfig*.json`, and test/build config
- environment templates such as `.env.example`
